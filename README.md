# 🚗 Automobile Sales Statistics Dashboard

This project is an interactive web dashboard built using Plotly Dash to analyze automobile sales data across recession and non-recession periods.

---

## 📊 Project Overview

The dashboard provides insights into automobile sales performance based on:
- Recession vs non-recession periods
- Year-wise sales trends
- Vehicle type performance
- Advertising expenditure
- Impact of unemployment rate

---

## 🧩 Key Features

### 📉 Recession Analysis
- Average automobile sales during recession years
- Sales comparison across vehicle types
- Advertising expenditure distribution by vehicle type
- Effect of unemployment rate on automobile sales

### 📈 Yearly Analysis
- Year-wise automobile sales trends
- Vehicle type-wise sales comparison
- Advertising expenditure analysis
- Unemployment impact on yearly sales

---

## ▶️ How to Run This Project Locally

Since this is a Dash application, it must be run locally.

### 1. Install dependencies
pip install dash pandas plotly

### 2. Run the application
python DV0101EN-Final-Assign-Part-2-Questions.py

### 3. Open in browser
http://127.0.0.1:8050/
