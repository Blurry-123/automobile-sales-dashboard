# 🚗 Automobile Sales Dashboard (Dash Project)

This project is an interactive web dashboard built using **Plotly Dash** to analyze automobile sales data.

---

## 📊 Features

- 📉 Recession period sales analysis
- 📈 Yearly automobile sales trends
- 🚗 Vehicle type comparison
- 📢 Advertising expenditure analysis
- 📊 Impact of unemployment rate on sales

---

## 🛠️ Technologies Used

- Python
- Dash
- Plotly Express
- Pandas
